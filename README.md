Перед началом работы, необходимо установить PostgreSQL 16 версии и выше.
При установке БД postgres необходимо поставить пароль postgres или же поменять в коде MainWindow.Xaml.cs строку password
Это необходимо для того чтобы создать БД для приложения. 
При запуске приложение проверяет существует ли БД и создает БД из таблиц и роль app из таблиц
После этого приложение работает со своим функционалом. БД сохраняется и приложение сохраняет все изменения.
